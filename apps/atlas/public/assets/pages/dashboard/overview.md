# Dashboard — Overview

## Purpose

The Overview page provides a high-level summary of key metrics, KPIs, and portfolio performance across all trading desks. It is the default landing page when navigating to the Dashboard app.

## Features

- **Key Metrics Cards** — At-a-glance view of total P&L, trade volume, open positions, and risk exposure
- **Performance Chart** — Daily/weekly/monthly performance trend line
- **Desk Summary Table** — Breakdown of metrics by trading desk
- **Alert Banner** — Highlights any breaches or anomalies requiring attention

## Access

Navigate to **Front Office → Dashboard → Overview** or directly via `/front-office/dashboard/overview`.

## Notes

- Data refreshes automatically every 30 seconds
- All figures are displayed in the user's configured base currency
- Click any metric card to drill down into the Analytics page
