# Dashboard — Analytics

## Purpose

The Analytics page provides interactive charts and data visualisations for analysing trading patterns, volumes, and trends over time. Users can explore data across multiple dimensions and time ranges.

## Features

- **Interactive Charts** — Line, bar, and area charts with zoom and pan
- **Time Range Selector** — Filter data by day, week, month, quarter, or custom range
- **Dimension Filters** — Slice data by desk, instrument type, currency, or counterparty
- **Comparison Mode** — Overlay multiple time periods for trend comparison
- **Export** — Download chart data as CSV or chart images as PNG

## Access

Navigate to **Front Office → Dashboard → Analytics** or directly via `/front-office/dashboard/analytics`.

## Notes

- Charts use real-time streaming data when available
- Large date ranges may take a few seconds to load
- Hover over data points for detailed tooltips
