# Dashboard — Reports

## Purpose

The Reports page allows users to generate, view, and export tabular reports on trading activity, P&L, and risk exposure. Reports can be customised with filters and saved for recurring use.

## Features

- **Report Templates** — Pre-built templates for common reports (Daily P&L, Trade Blotter, Risk Summary)
- **Custom Filters** — Filter by date range, desk, instrument, counterparty, and status
- **Sortable Columns** — Click column headers to sort ascending/descending
- **Pagination** — Navigate through large result sets
- **Export Options** — Download as CSV, Excel, or PDF

## Access

Navigate to **Front Office → Dashboard → Reports** or directly via `/front-office/dashboard/reports`.

## Notes

- Saved report configurations persist across sessions
- Scheduled reports can be set up via the Settings app
- Maximum export size is 100,000 rows
