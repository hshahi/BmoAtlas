# Stocks — Breakdown

## Purpose

The Breakdown page provides a detailed view of stock positions with both table and list presentation modes. Users can filter, sort, and drill down into individual positions.

## Features

- **Dual View Mode** — Toggle between table view and card/list view
- **Position Details** — Quantity, average price, current value, unrealised P&L
- **Filtering** — Filter positions by sector, exchange, currency, or P&L status
- **Sorting** — Sort by any column (name, value, change, P&L)
- **Drill-Down** — Click a position to see detailed trade history

## Access

Navigate to **Front Office → Stocks → Breakdown** or directly via `/front-office/stocks/breakdown`.

## Notes

- Position data refreshes in sync with the Summary page's streaming feed
- The table view supports column reordering via drag-and-drop
- Export to CSV is available from the table view toolbar
