# Settings — Profile

## Purpose

The Profile page allows users to view and edit their user profile, contact details, role assignments, and access permissions.

## Features

- **Personal Information** — Name, email, phone number, and department
- **Role & Permissions** — View assigned roles and access levels (read-only)
- **Desk Assignments** — See which trading desks you have access to
- **Activity Log** — Recent login history and session information
- **Avatar** — Upload or change your profile picture

## Access

Navigate to **Front Office → Settings → Profile** or directly via `/front-office/settings/profile`.

## Notes

- Role and permission changes must be requested through your line manager
- Profile changes are audited and logged
- Two-factor authentication settings are managed separately via IT Security
