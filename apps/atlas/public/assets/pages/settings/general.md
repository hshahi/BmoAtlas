# Settings — General

## Purpose

The General settings page provides application-wide configuration options including language, timezone, notification preferences, and display options.

## Features

- **Language** — Select the UI language (English, French, German)
- **Timezone** — Set your local timezone for date/time display
- **Notifications** — Configure email and in-app notification preferences
- **Display Options** — Choose number format, date format, and decimal precision
- **Theme** — Select light, dark, or system-auto theme (also available via toolbar toggle)

## Access

Navigate to **Front Office → Settings → General** or directly via `/front-office/settings/general`.

## Notes

- Changes are saved automatically
- Some settings may require a page refresh to take effect
- Organisation-level defaults are set by administrators
